# playwright-bdd-framework
# Playwright BDD Automation Framework - Mahesh D

## 🚀 Tech Stack
- Playwright (UI Automation)
- Cucumber (BDD)
- TypeScript
- Allure Reporting
- GitHub Actions (CI/CD)
- Jira Integration

## 📂 Structure
- pages → Page Object Model
- tests → Features + Steps
- api → API validation layer
- utils → Helpers (Jira, reporter)

## ▶️ Run Tests
npm test

## 📊 Generate Report
npm run report

## 🔁 CI/CD
Triggered on every push via GitHub Actions

## 🔗 Features
- Hybrid UI + API validation
- BDD readable scenarios
- Scalable architecture
